16. Personalización (Configuración general)

Sylpheed ofrece muchas posibilidades de configurar su interfaz para adaptarse a sus gustos personales.

16.1 Ventana de Configuración general

Para realizar cambios en la interfaz de Sylpheed es necesario abrir la opción "Preferencias comunes" en el menu de Configuración.

Aparecerá un cuadro de diálogo con diversas pestañas que le permitirán configurar Sylpheed.

16.2 Descripción de Cada Entrada

En la tercera pestaña, Componer, se pueden cambiar las siguientes configuraciones:

  • Si se desea citar el texto original de un mensaje.
  • El carácter que será colocado delante de cada línea de texto citada.
  • El texto que aparecera encabezando el texto citado.
  • El separador para la firma (opcional), el grupo de caracteres que se coloca delante del texto de la firma.
  • Si desea o no insertar la firma automáticamente.
  • Definir si la cuenta que recibió el mensaje debe ser usada automáticamente como la cuenta desde la que responder al mensaje. Esto también influye en la firma que se seleccionará.
  • La posición de la línea a partir de la cual se debe recortar el texto.
  • Si el texto citado deberá ser también recortado.
  • Indicar si el texto del correo debe ser recortado o no. Si escoge no recortar, las líneas no serán truncadas por caracteres de retorno de carro.
En la cuarta pestaña, Ver, se pueden habilitar o no las siguientes opciones:

  • Fuente Se pueden establecer las fuentes que se usarán para visualizar los mensajes.
  • Traducir cabeceras Esta opción permite que aparezcan en castellano los nombres de las cabeceras más comunes.
  • Ver el número de no leídos Habilita que aparezca el número de mensajes sin leer que hay en cada carpeta al lado del nombre.
  • Mostrar destinatario Con esta opción Sylpheed mostrará el destinatario en la columna del remitente en el caso de que el que haya enviado el mensaje sea usted mismo.
  • Habilitar barra de desplazamiento Permite que aparezca o no la barra de desplazamiento debajo del resumen de cabeceras. Si esta deshabilitada no podrá desplazar el contenido de la ventana para ver los asuntos de los mensajes si son demasiado largos.
  • Expandir hilos Si esta deshabilitada esta opción y esta habilitada la vista jerárquica de mensajes los hilos aparecerán colapsados con un signo '+' delante. Si se habilita los hilos se desplegarán automáticamente.
  • Ver mensajes no leídos en negrita Con esta casilla los mensajes del resumen de cabeceras que aún no hayan sido leídos apareceran con una fuente más gruesa. Deshabilitando la opción todos aparecerán con la misma fuente.
  • Formato de fecha Esta opción permite cambiar la manera en que aparece la fecha en los mensajes de correo. Pulsando el botón '...' se abrirá una nueva ventana informativa con los códigos que se pueden utilizar para dar formato a la fecha. A medida que se cambie la cadena de formato se mostrará debajo una muestra del aspecto que tomará la fecha.
  • Elementos visibles Este botón permite especificar que elementos aparecerán en la vista resumen de las cabeceras.
Al lado de estas opciones está la pestaña de Mensaje, se pueden encontrar las siguientes opciones:

  • Permitir colores Como su nombre indica, permite definir que colores se utilizarán para ver el texto, el texto citado y las URLs.
  • Mostrar alfabetos de 2 bytes con 1 carácter. Esta opción es necesaria para lenguajes codificados en caracteres de 16 bits, como el Japonés y el Chino.
  • Mostrar el panel de cabeceras encima del mensaje. Habilitada, esta opción muestra las cabeceras Desde, Para y Asunto del correo seleccionado en un area separada encima del texto del mensaje.
  • Mostrar cabeceras breves en la vista del mensaje. Esta opción permite seleccionar que cabeceras deben mostrarse en la ventana del mensaje. Hay opciones para habilitarlas todas, ninguna o solo las selecciondas.
  • Interlineado. Esta opción permite seleccionar cuantos píxels tiene que dejar Sylpheed entre cada linea de texto en la ventana del mensaje.
  • Dejar espacio de cabecera. Marcando esta casilla se hará que el comienzo de las nuevas líneas (sin partir, donde el autor haya pulsado la tecla Intro/Retorno) sea indentado unos pocos píxels. Puede ser útil para leer correos grandes.
  • Desplazar media página. Cuando este activa, el texto de la ventana de mensaje solo se desplazará media página, en vez de una página completa, al pulsar la barra espaciadora.
  • Desplazamiento suave. Habilitando esta casilla hará que el texto en la ventana de mensaje se desplaze mas suavemente, deslizándose en vez de saltar. Puede ser más agradable a la vista, es una cuestión de gustos personales.
    • La pestaña "Interfaz" permite incluso una mayor personalización de Sylpheed. Esta pestaña contiene:
      • Abrir el primer mensaje no leído al abrir una carpeta.
      • Abrir la carpeta de entrada automáticamente despues de recibir correo nuevo.
      • Ejecución inmediata de comandos tales como borrar, mover o copiar.
      • Influenciar la manera en que aparece el diálogo de recepción cuando se recupera el correo nuevo (Siempre, Cuando hay una ventana activa, Nunca).
      • Doble-click sobre una dirección de correo puede ser usado para poner esa dirección en un nuevo correo.
      • Confirmar si se quiere salir de Sylpheed.
      • Limpar la papelera al salir de Sylpheed.
      • Hacer que Sylpheed pregunte si se quiere vaciar la papelera antes de vaciarla realmente.
      • Avisar, cuando se esta cerrando Sylpheed, que hay mensajes en la cola para enviar.
      Finalmente existe la pestaña "Otras" donde se pueden definir los programas que se quieren utilizar como aplicaciones de ayuda a Sylpheed.
      • Navegador web. Aquí se puede introducir el nombre de tu navegador web favorito (Netscape, Opera, Mozilla, Lynx etc.). Añade el parámetro %s al comando para que Sylpheed sepa donde tiene que proporcionar el texto de la URL pulsada que el navegador tiene que visitar.
      • Impresión. Aquí se puede definir el programa que se debe usar para imprimir. El predeterminado es lpr, y la %s es el nombre del fichero temporal que Sylpheed crea para que lpr imprima.
      • Editor externo. Se puede seleccionar un editor externo con el que escribir los correos, por ejemplo vi, emacs o gedit. Hay que recordar que para editores en modo texto como vi, hace falta definir la ventana terminal donde se van a ejecutar, p.ej. xterm -e vi %s. Si no el editor no tendrá ningún lugar donde poder aparecer. Aquí el parámetro %s es de nuevo el nombre del fichero temporal que Sylpheed crea para recoger el texto escrito.
