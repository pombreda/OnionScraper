4. Instalación.

4.1 Entorno requerido

Para compilar Sylpheed con éxito, necesitas por lo menos:

  • Cualquier UNIX o sistema operativo similar que sea compatible con POSIX, por ejemplo Linux, FreeBSD, Solaris
  • GTK+ 1.2.6 o posterior
  • Un compilador ANSI C reciente (gcc 2.7.2.3 también debería servir) Nota: se ha reportado que el Sun C no compila Sylpheed.
Opcionalmente necesitas:
  • Imlib o gdk-pixbuf (para la vista incorporada de imágenes)
  • libcompface (para X-Face)
  • libjconv (para internacionalización)
  • GPGME (para GnuPG)
  • OpenLDAP (para la conexión LDAP)
  • Jpilot (para utilizar la agenda de direcciones Palm)
  • el equivalente de GTK+-devel y Xfree86-devel. (Sino ./configure fallará.)
Aviso: Sylpheed puede ser inestable dependiendo del entorno. Por favor utiliza si es posible la versión más reciente de las librerías.

4.3 Detalles de la instalación

  • Descargar la última versión de las fuentes de Sylpheed desde http://sylpheed.good-day.net/
  • Descomprime los ficheros tar con las fuentes a un directorio nuevo
  • tar -xvfz sylpheed-.tar.gz (para tar.gz)
  • tar -xvfI sylpheed-.tar.bz2 (para tar.bz2)
  • cd al nuevo directorio creado sylpheed-
  • Ejecuta ./configure para verificar tu entorno y crear los archivos make de la instalación
  • Ejecuta el programa 'make' para compilar las fuentes. No realices este paso si existen errores fatales en el paso anterior. Los errores fatales indican que falta cosas para compilar Sylpheed adecuadamente.

Si el proceso make se ha completado sin error, ejecuta "make install" (como root) para instalar el programa y los archivos de soporte adicionales.

4.4 Instalación en Linux utilizando RPM

Como root, ejecuta el comando "rpm -ivh" en el archivo RPM que hayas descargado. Tan pronto como éste proceso termine ya puedes ejecutar Sylpheed como usuario.

4.5 Instalación en Debian Linux

Con conexión a internet:

apt-get install sylpheed (para instalarlo con conexión internet)

Sin conexión a internet, descarga el paquete debian, por ejemplo en http://packages.debian.org/unstable/mail/sylpheed.html

dpkg -i sylpheedx.x.x-1i386.deb

dpkg --install sylpheedx.x.x-1i386.deb

4.6 Instalación en FreeBSD

Utilizando la herramienta /stand/sysinstall (interface basada en menúes):

ejecuta "/stand/sysinstall" como root

en el menú principal elige "Configure" -> "Packages" -> "mail", entonces "sylpheed", luego de regreso a la lista "Packages" , elige "Install"

Utilizando portes

Consigue el porte de sylpheed en:

ftp://ftp.freebsd.org/pub/FreeBSD/ports/i386/packages-4.3-release/mail

luego, como root: pkg-add sylpheed-0.x.x.tgz

utilizando los portes para compilar:

primero, actualiza tu porte, descargalo http://www.freebsd.org/ports/mail.html

cd /usr/ports/mail/sylpheed

make

Si ésto funciona, ejecuta "make install"

En el caso de que falle, descarga el paquete requerido si es necesario (en la misma página), coloca el paquete en /usr/ports/distfiles/ luego :

  • cd /usr/ports/mail/sylpheed
  • make
  • make install

¿Necesitas una versión más actualizada?

Simplemente utiliza el método estándar de Unix (consulta "Detalles de la instalación")
