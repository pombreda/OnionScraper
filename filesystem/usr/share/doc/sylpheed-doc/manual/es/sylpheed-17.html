17. Referencia

En esta sección encontrará alguna información que podría ser de ayuda para hacer más claro el uso de Sylpheed respecto a los atajos del teclado utilizados, así como una visión del formato del libro de direcciones.

17.1 Lista de atajos de teclado

Atajos de una sola tecla:

  • a: crea un mensaje nuevo
  • d: borra el mensaje seleccionado
  • f: reenvia el mensaje seleccionado
  • g: ir a una carpeta específica mediante el diálogo de selección de carpetas
  • n: ir al siguiente mensaje o parte de un mensaje multipartes
  • p: ir al mensaje previo o a la seccion previa de un mensaje multipartes
  • v: aparecer o desaparecer la vista de mensaje
  • w: lo mismo que a
  • y: abre el diálogo "Salvar como..."

Atajos para el menú Fichero:

  • Alt-F: abrir el menu Fichero
  • Alt-P: imprimir
  • Alt-W: cerrar
  • Alt-Q: salir

Atajos para el menú Editar:

  • Alt-E: abrir el menu Editar
  • Ctrl-C: copiar
  • Ctrl-A: seleccionar todo
  • Ctrl-S: buscar

Atajos para el menú Ver:

  • Alt-V: abrir el menú Ver

Atajos para el menú Mensaje:

  • Alt-M: abrir el menu Mensaje
  • Alt-I: comprobar el correo en la cuenta actual
  • Shift-Alt-I: comprobar el correo en todas las cuentas
  • Alt-N: componer un mensaje nuevo
  • Alt-R: responder al mensaje
  • Ctrl-Alt-R: responder al remitente (lo contrario de responder a p.ej. a una lista de correo).
  • Shift-Alt-R: responder a todos (incluyendo todas las direcciones en Desde: y Cc:)
  • Ctrl-F: reenviar el mensaje
  • Shift-Ctrl-F: reenviar el mensaje como adjunto
  • Alt-O: mover el mensaje
  • Alt-D: borrar el message
  • Shift-Ctrl-N: abrir el mensaje en una nueva ventana
  • Ctrl-U: ver el mensaje en formato fuente
  • Ctrl-H: ver todas las cabeceras del mensaje

Atajos al menú Resumen

  • Alt-S: abrir el menú Resumen
  • Alt-X: ejecutar (comandos pendientes, solo es útil cuando los comandos no se ejecutan inmediatamente)
  • Alt-U: actualizar vista
  • Alt-G: ir a otra carpeta
  • Ctrl-T: activar vista jerárquica
  • Shift-Ctrl-T: vista plana (desactiva la vista jerárquica)

Atajos al menu Herramientas

  • Alt-T: abrir el menu Herramientas
  • Alt-A: abrir el libro de direcciones
  • Alt-L: abrir el fichero de traza

17.2 Formato de datos del libro de direcciones

El libro de direcciones es un fichero localizado en el directorio ~/.sylpheed, cuyo nombre es addressbook.xml. El formato general del libro de direcciones es el siguiente: 

< addressbook ><p>
 < common_address ><p>
    < group name="Direcciones comunes" ><p>
    < item ><p>
     < name > Paul < /name ><p>
     < address > paul@nlpagan.net < /address ><p>
     < remarks > Escribe el manual de Sylpheed < /remarks ><p>
        < /item ><p>
    < /group ><p>
 < /common_address ><p>
< /addressbook >
