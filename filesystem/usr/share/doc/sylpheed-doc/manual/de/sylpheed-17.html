17.2 Aufbau des Adressbuches

Das Adressbuch befindet sich im Verzeichnis  /.sylpheed. Die Datei heißt addressbook.xml. Das allgemeine Format von Einträgen im Adressbuch ist: 

< addressbook ><p>
 < common_address ><p>
    < group name="Allgemeine Adresse" ><p>
    < item ><p>
     < name > Paul < /name ><p>
     < address > paul@nlpagan.net < /address ><p>
     < remarks > Schreibt Sylpheed Handbuch < /remarks ><p>
        < /item ><p>
    < /group ><p>
 < /common_address ><p>
< /addressbook >
