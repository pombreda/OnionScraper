5. Démarrer et quitter Sylpheed

5.1 Démarrer Sylpheed

Démarrer Sylpheed peut être fait en tapant sylpheed dans le prompt d'un terminal et en pressant Entrée. Vous pouvez aussi ajouter Sylpheed à un menu dans KDE ou Gnome, ou lui créer un raccourci sur le bureau si votre gestionnaire de fenêtres le permet.

Lorsque vous démarrez Sylpheed, le programme vous demandera où vous souhaitez stocker les courriers que vous allez télécharger. Vous pouvez accepter la valeur par défaut ($HOME/Mail) ou configurer votre propre répertoire de courriers.

Sylpheed chargera alors ses configurations par défaut. La manière de les changer est décrite plus loin dans ce manuel.

5.2 Quitter Sylpheed

Quitter Sylpheed peut se faire de différentes manières :

  • dans le menu Fichier, sélectionner Quitter ;
  • presser « Ctrl-Q » ;
  • ou cliquer dans la barre de titre sur le bouton qui ferme le programme.
