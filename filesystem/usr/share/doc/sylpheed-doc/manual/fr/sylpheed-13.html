13. Importation et exportation de courriers

13.1 Importer des messages au format MBOX

Sylpheed vous permet d'importer des messages au format MBOX à partir du menu Fichier. Mais d'abord, vous devez créer un dossier dans lequel vous souhaitez que les messages soient installés. Ensuite, sélectionnez Fichier -> Importer un fichier mbox.... Utilisez la fonction de navigation pour sélectionner le fichier MBOX et le dossier destinataire et pressez Ok. Et voilà, vos courriers sont importés.

C'était facile, n'est ce pas ?

13.2 Importer depuis les clients de messageries sous Windows (Becky!, AL-Mail, Outlook Express4, etc.)

Pour pouvoir importer des répertoires de courriers de certains clients de messageries fonctionnant sous Windows, vous pouvez utiliser le programme mbx2mbox situé à l'adresse http://mbx2mbox.sourceforge.net.

13.3 Exporter le format MH de Sylpheed vers un fichier MBOX

Sylpheed offre la possibilité d'exporter un de ses dossiers vers le format standard MBOX. Sélectionnez pour cela le menu Fichier -> Exporter vers un fichier mbox..., puis sélectionnez le dossier à exporter et enfin choisissez le nom du fichier à générer.
