6. Les options du menu de Sylpheed

Le menu de Sylpheed est le point d'entrée à toutes les opérations que vous pouvez réaliser avec Sylpheed. La structure du menu est la suivante :

6.1 Menu Fichier

  • Ajouter une boîte aux lettres... : cette option vous permet d'ajouter un nouveau répertoire de courrier MH. Il générera automatiquement dans la nouvelle boîte aux lettres les dossiers des courriers reçus (Réception), envoyés (Envoyés), en attente d'envoi (File d'attente), brouillons (Brouillon) et des courriers supprimés (Corbeille).
  • Mise à jour de l'arbre des répertoires : il relit l'arbre complet des boîtes aux lettres existantes pour qu'elles reflètent l'état correct (ce n'est pas nécessaire habituellement).
  • Dossiers : vous pouvez y ajouter, renommer ou effacer des dossiers de courrier.
    • Créer un nouveau dossier... : Créer un nouveau dossier! Avant d'utiliser cette option vous devez sélectionner un dossier, parmi ceux de l'arbre des dossiers, où sera créé votre nouveau dossier.
    • Renommer le dossier... : Permet de changer le nom du dossier sélectionné.
    • Supprimer le dossier : Supprime le dossier sélectionné ainsi que tous les messages qu'il contenait.
  • Importer un fichier mbox... : vous pouvez importer des fichiers mbox (comme ceux créés par Pine) dans une boîte aux lettres MH.
  • Exporter vers un fichier mbox... : vous pouvez exporter une boîte aux lettres MH vers un fichier mbox.
  • Vider la corbeille : évidemment, ceci videra le dossier de la poubelle.
  • Sauvergarder comme... : vous pouvez sauvegarder les messages sélectionnés dans un fichier texte pur sur le disque dur.
  • Imprimer... : vous pouvez imprimer les messages sélectionnés vers une imprimante.
  • Quitter : arrête Sylpheed.

6.2 Menu Édition

  • Copier : vous pouvez copier le texte sélectionné vers un tampon, à partir duquel vous pouvez le coller dans une zone différente. Raccourci : Ctrl-C
  • Tout sélectionner : sélectionne tout le texte dans la fenêtre courante. Raccourci : Ctrl-A
  • Chercher dans le message... : cherche dans la fenêtre courante un texte que vous choisissez. Raccourci : Ctrl-F
  • Chercher dans le dossier... : recherche un message selon le contenu des champs A, De, Sujet, champs d'en-tête ou corps du message. La recherche commence à partir du message sélectionné dans le dossier sélectionné. Raccourci : Shift-Ctrl-F

6.3 Menu Vue

Le menu Vue vous permet de changer la manière dont Sylpheed apparaît. Vous pouvez sélectionner un jeu différent de fenêtres séparément (vue de dossiers, messages, etc.), valider ou désactiver la barre d'état et ainsi de suite. Il contient aussi les commandes pour activer/désactiver l'affichage de certaines parties du message courant, les outils pour trier les messages du dossier sélectionné selon différents critères.

  • Afficher ou cacher : vous permet de choisir les parties de l'interface utilisateur de sylpheed que vous voulez afficher.
    • Arbre des répertoires : pas encore implémenté.
    • Vue des message : Montre ou cache la fenêtre des messages. Lorsqu'elle est cachée, les messages sont affichés dans une fenêtre séparée. Pour lire un message double-cliquez dessus dans la fenêtre de résumé. Le message est alors oouvert dans une nouvelle fenêtre. Vous pouvez fermer cette derbière en appuyant sur la touche Echap.
    • Barre d'outils : configure L'apparence de la barre d'outils : avec icône et texte ou l'un des deux.
      • Icônes et textes : la barre d'outils est affichée avec les icônes et leur texte.
      • Icônes seulement : la barre d'outils est affichée avec les icônes seulement.
      • Textes seulement : la barre d'outils est affichée avec le texte, mais sans les icônes.
      • Pas de barre d'outils : la barre d'outils n'est pas affichée.
    • Barre d'état : la montre ou la cache.
  • Arbre de répertoire séparé : sélectionne entre avoir l'arbre des répertoires attaché à la fenêtre principale ou le laisser flotter séparément.
  • Séparer la vue du message : permet le choix entre avoir le message sélectionné du cadre du sommaire dans la fenêtre principale ou l'avoir dans une fenêtre flottante séparée.
  • Trier : permet l'affichage trié des messages du dossier sélectionné. Chaque option effectue le tri par ordre décroissant et peux être inversée en choisissant à nouveau la même option du menu (l'ordre s'inverse à chaque fois que l'on choisi la même option du menu).
    • Trier par numéro : trie les messages sur la base de leur numéro MH.
    • Trier par taille : trie les messages selon leur taille.
    • Trier par date : trie les messages en fonction de leur date.
    • Trier par expéditeur : trie les messages dans l'ordre alphabétique sur l'adresse de l'expéditeur (champ De).
    • Trier par sujet : trie les messages par ordre alphabétique du sujet.
    • Trier par couleur : trie les messages selon leur couleur. Lisez l'aide du menu contextuel plus bas pour en savoir davantage sur la manière d'affecter une couleur à un message.
    • Trier par marque : sépare les messages marqués de ceux non marqués. Lisez l'aide du menu contextuel pour savoir comment marquer un message.
    • Trier par non lu : sépare les messages lus de ceux non lus.
    • Trier par piece jointe : sépare les messages qui ont une ou plusieurs pièces jointes de ceux qui n'en n'ont pas.
    • Ne pas trier : ne trie pas les messages.
    • Ordre croissant : le tri des messages s'effectue par ordre croissant. Lorsque vous sélectionnez cette option, une petite flèche s'affiche dans la colonne triée reflétant ce choix.
    • Ordre décroissant : le tri des messages s'effectue par ordre décroissant.
    • Attirer par sujet : place les messages ayant des sujets similaires ou identiques les uns à cotés des autres.
  • Vue par fil de discussion : active ou désactive la vue par fil de discussion. Lorsque l'option est activée, tous les messages d'une même discussion (discussion générée par les réponses successives aux messages) sont représentés dans un arbre représentant cette discussion. Chaque branche de l'arbre peut être ouverte ou fermée en utilisant le plus (+) ou le moins (-) qui la compose. Les fils de discussion sont basés sur les champs In-Reply-To et Reference de l'en-tête des messages. Ainsi, ne commencez jamais une nouvelle conversation en répondant à un ancien message, cela aurait pour effet de raccrocher votre message à la branche de l'ancienne conversation, sans rapport avec la nouvelle. Raccourci : Ctrl-T
  • Déployer les fils de discussion : lorsque la vue par fil de discussion est activée, cette option permet de développer tous les fils de discussion du dossier ouvert.
  • Compacter les fils de discussion : lorsque la vue par fil de discussion est activée, cette option permet de réduire tous les fils de discussion du dossier ouvert
  • Choix des éléments affichés... : utilisez cette option pour choisir les éléments à afficher dans la fenêtre résumé. Vous pouvez également changer ces paramêtres dans les préférences générales.
  • Aller à : permet la navigation d'un message à un autre grâce aux différents critères du sous menu.
    • Message précédent : va au message précédent du dossier ouvert en cours. Raccourci : P
    • Message suivant : va au message suivant du dossier ouvert en cours. Raccourci : N
    • Non lu précédent : va au message non lu précédent du dossier ouvert en cours. Raccourci : Shift-P
    • Non lu suivant : va au message non lu suivant du dossier ouvert en cours. Raccourci : Shift-N
    • Message nouveau précédent : va au nouveau message précédent du dossier ouvert en cours.
    • Message nouveau suivant : va au nouveau message suivant du dossier ouvert en cours.
    • Marqué précédent : va au message marqué précédent du dossier ouvert en cours.
    • Marqué suivant : va au message marqué suivant du dossier ouvert en cours.
    • Colorié précédent : va au message colorié précédent du dossier ouvert en cours.
    • Colorié suivant : va au message colorié suivant du dossier ouvert en cours.
    • Autre dossier... : chane le dossier ouvert en cours. Choisissez dans l'arbre des dossier de la nouvelle fenêtre qui s'affiche le dossier que vous voulez ouvrir.
  • Jeux de caractères : vous pouvez sélectionner le jeu de caractères que vous voulez utiliser pour vos messages. Ce jeu de caractères sera utilisé lorsque vous enverrez vos messages. Notez que Sylpheed essaiera toujours d'utiliser le jeu de caractères le plus compatible avex les carcatères utilisés dans votre message, ainsi, a moins que vous n'utilisiez des caractères spéciaux du jeu de caractère latin-1 dans votre message (NDT : les accents), le jeu sélectionné sera défini comme étant du 7bits ASCII même si vous avez sélectionné le jeu de caractères ISO-8859-1. L'option d'auto détection laisse Sylpheed détecter automatiquement le jeu de caractères à utiliser en fonction des caractères contenus dans les messages.
  • Ouvrir dans une nouvelle fenêtre : ouvre le message sélectionné dans une nouvelle fenêtre. Cette nouvelle fenêtre peut être fermée en pressant la touche Echap. Raccourci : Ctrl-Alt-N
  • Voir le source du message : ouvre une nouvelle fenêtre et affiche le source complet du message (sans aucune mise en forme et avec toutes les en-têtes). Raccourci : Ctrl-U
  • Afficher tous les en-têtes : Active ou désactive l'affichage, dans le message, de la totalité des en-têtes. Raccourci : Ctrl-H
  • Mettre à jour : met à jour la fenêtre du résumé. Raccourci : Ctrl-Alt-U

6.4 Menu Message

Ici, vous pouvez trouver les options suivantes :

  • Relever le courrier : ceci permet de collecter tous les nouveaux courriers dans votre boîte aux lettres personnelle. Raccourci : Ctrl-I
  • Relever le courrier de tous les comptes : Sylpheed scrutera toutes les identités de comptes définies pour les nouveaux courriers. Raccourci : Shift-Ctrl-I
  • Interrompre la relève du courrier : arrête la réception des nouveaux messages.
  • Envoyer les messages en file d'attente : envoie tous les messages que vous avez mis dans le dossier File d'attente.
  • Composer un nouveau message : vous pouvez écrire un nouveau courrier.
  • Répondre : un courrier sera créé avec l'expéditeur du message sélectionné dans le champ du destinataire. En fonction de la configuration, votre signature sera ajoutée au courrier et le texte de courrier original sera décalé.
  • Répondre à tous : comme Répondre mais les destinataires seront toutes les adresses du courrier original (incluant ceux dans la partie cc:).
  • Répondre à l'auteur : comme Répondre mais l'adresse du destinataire est celle de l'en-tête "De" et non celle de l'en-tête "Répondre à".
  • Transférer : un message sera créé avec le message original dans le corps du texte. Vous pouvez alors entrer une adresse de destinataire, éditer le texte si vous le souhaitez et l'envoyer.
  • Transférer comme attachement : comme Transférer mais le texte du message sera envoyé attaché comme un fichier texte séparé.
  • Rediriger : envoyer à nouveau le message sélectionné à quelqu'un d'autre. En comparaison avec un transfert, le message n'est pas marqué comme ayant été transféré et l'auteur est l'auteur du message original.
  • Rééditer : vous laisse ouvrir n'importe quel message déjà envoyé en mode d'édition (NDT : donc de le modifier) afin de l'expédier à nouveau.
  • Déplacer : vous pourrez déplacer le message vers un répertoire différent. Vous pouvez sélectionner le répertoire de destination depuis la fenêtre de navigation.
  • Copier : comme pour Déplacer, seul le message est copié dans le nouveau répertoire, au lieu d'être déplacé.
  • Supprimer : vous pouvez effacer le message qui est sélectionné dans le tableau du sommaire.
  • Marquer : Vous obtenez un sous-menu : Marquer, Démarquer, Marquer comme lu, Démarquez comme lu. Lorsque vous marquez les messages, vous pouvez faire des actions spécifiques (i.e. déplacer, effacer) sur tous les messages marqués. Vous pouvez démarquer les messages que vous avez marqués par erreur. Les autres options font ce qu'indiquent leur nom, elles marquent ou démarquent un message comme lu.

Note sur l'utilisation des fonctions Déplacer, Copier et Supprimer : si vous n'avez pas choisi d'exécuter immédiatement les commandes dans les options de configuration, vous devrez sélectionner la commande "Exécuter". Il s'agit d'un bouton de la barre d'outils qui se trouve également dans le menu Outils.

6.5 Menu Outils

Dans le menu Outils se trouvent les fonctions de control de filtres, le fonctions du carnet d'adresse et la fenêtre permettant le contrôle des logs.

  • Carnet d'adresse : ouvre la fenêtre du carnet d'adresse. Raccourci : Shift-Ctrl-A
  • Ajouter l'auteur dans le carnet d'adresse : ajoute l'auteur du message sélectionné à l'un de vos carnet d'adresse. Une fenêtre s'ouvre pour que vous choisissiez le carnet d'adresse dans lequel vous voulez sauvegarder l'adresse de l'auteur du message.
  • Filtrer les messages : Applique les règles de filtrage aux messages de la boîte aux lettres de Réception.
  • Créer une règle de filtrage : crée une nouvelle règle basée sur le message sélectionné.
    • Automatiquement : ouvre la fenêtre de création de règles pour définir une nouvelle règle.
    • Avec "De" : ouvre la fenêtre de création de règles et rempli le champ "de" du filtre afin de construire un filtre basé sur l'auteur du message sélectionné.
    • Avec "A" : ouvre la fenêtre de création de règles et rempli le champ "A" du filtre afin de construire un filtre basé sur le destinataire du message sélectionné.
    • Avec "Sujet" : ouvre la fenêtre de création de règles et rempli le champ "sujet" du filtre afin de construire un filtre basé sur le sujet du message sélectionné.
  • Supprimer les messages en double : cette option détruit les messages du dossier en cours qui vont ont été envoyés en double. Par exemple vous recevez un message directement and via une liste de diffusion dans le même temps.
  • Actions : ce menu est vide jusqu'a ce que vous définissiez des actions (voyez l'aide du menu Configuration ci dessous).
  • Exécuter : exécute toutes les commandes Déplacer, Supprimer et Copier en attente. Seulement valable si l'option d'exécution immédiate est désactivée dans les préférences générales. Raccourci : X
  • Fenêtre de log : ouvre la fenêtre de log. Dans cette fenêtre vous pouvez voir ce que fait Sylpheed. Ces traces vous permettent de contrôler ce qui ce passe quand vous suspectez que quelque chose ne vas pas. Vous pouver copier-coller les informations dans un message électronique et l'envoyer aux développeurs ou à la liste de diffusion.

Dans la fenêtre de log (journal de bord), vous pouvez voir ce que Sylpheed fait. Garder une trace des choses de cette manière est important lorsque vous suspectez des anomalies. Vous pouvez coller les informations de la fenêtre de log dans un courrier et l'envoyer aux développeurs dans la liste de diffusion.

6.6 Menu Configuration

Dans le menu Configuration, clairement, vous pouvez configurer Sylpheed de la manière dont vous voulez qu'il travaille et réagisse.

  • Préférences générales... : dans cette option, il vous sera présenté un dialogue dans lequel vous pouvez sélectionner un certain nombre d'onglets. Chaque onglet représente un jeu spécifique de configurations dans Sylpheed. Dans le chapitre suivant, ces options de dialogue sont expliquées avec plus de précision.
  • Options de filtrage... : cette option vous montre un dialogue dans lequel vous pouvez configurer les règles de filtrage que Sylpheed appliquera à la réception des courriers. Vous spécifiez d'abord quelle(s) partie(s) de votre courrier Sylpheed doit contrôler et ensuite vous indiquez à Sylpheed quoi faire. Vous pouvez déplacer les messages vers un répertoire différent (incluant la corbeille pour se débarrasser des spams). Le dossier de destination est sélectionnable dans une fenêtre de navigation qui apparaît lorsque vous pressez le bouton Sélection...

    Vous pouvez aussi choisir de ne pas récupérer les courriers. Notez que cela laissera le message sur le serveur de courrier de votre prestataire, le message n'y sera pas détruit !

    Lorsque vous pressez le bouton Enregistrer, la règle de filtrage est ajoutée à la liste des règles. Si vous sélectionnez Remplacer, alors la règle sélectionnée est remplacée par l'information des configurations entrées. Enfin, le bouton Supprimer permet de détruire les règles de filtrage sélectionnées.

    Notez que vous pouvez configurer deux conditions pour une même règle. Elles peuvent être combinées par l'une des opérations logiques ou ou et. L'opération ou signifie qu'au moins une des deux conditions doit être satisfaite pour que la règle soit exécutée. L'opération et signifie que les deux conditions doivent être satisfaites simultanément pour que l'action soit effectuée.

    À droite de la liste des règles se trouvent les boutons appelés Plus haut et Plus bas. Grâce à eux, vous pouvez déplacer dans la liste les règles vers le haut ou vers le bas pour les réordonner. Ceci affectera l'ordre d'exécution des règles et a en générale une incidence sur le résultat final du filtrage, veuillez donc prendre en compte l'ordre dans lequel vous arrangez les règles.

  • Modèles... : Vous permet la création d'un nouveau modèle de message. Lorsque vous composez un nouveau message vous pouvez utiliser l'un des modèles que vous avez définis. Les modèles peuvent contenir des caractères spéciaux qui sont remplacés, afin de créer un texte dynamique, quand le modèle est utilisé. Cliquez sur le bouton Symboles pour voir les caractères spéciaux disponibles.
  • Actions... : vous permet la création de nouvelles commandes qui seront alors ajoutées au menu 'Outils/Actions'. Une Action est une commande qui s'exéctue sur une partie du message, le corps du message ou le fichier complet du message. L'exécution de cette commande peut se produire en arrière plan lorsqu'elle est utilisée comme un filtre (elle lit, modifie et remplace un message).
  • Configuration du compte actuel... : ici, vous pouvez configurer les options du compte que vous utilisez actuellement, par exemple pour décider si le courrier doit être effacé après traitement, ou de quel serveur il provient, etc. Cette fenêtre de configuration est la même que celle utilisée lors de la création d'un nouveau compte.
  • Création d'un nouveau compte... : ici vous donnez un nom au nouveau compte à créer (chaque compte possède un nom reconnaissable), vous configurez votre nom et une adresse de courrier, etc. Vous configurez le type de serveur pour lequel vous avez un accès (POP3, IMAP4, boîte de courrier local, serveur de nouvelles) et où doivent aller les messages téléchargés (le répertoire par défaut est inbox). Dans l'onglet Réception vous pouvez activer ou désactiver certaines options, dans l'onglet Envoi, vous pouvez configurer la manière d'envoyer vos messages. Dans l'onglet Composition, vous pouvez désigner un fichier de signature spécifique et dans Avancée, vous pouvez configurer explicitement les ports pour les serveurs utilisés. (Ne vous compliquez la vie pas avec ceux-ci, à moins que vous ne soyez sûr d'être obligés de le faire !!)
  • Édition des comptes... : ici, vous pouvez sélectionner à partir d'une liste un des comptes que vous avez créés pour en modifier la configuration. Notez que si vous êtes en train de composer un message vous ne pouvez pas éditer le compte que vous utilisez pour ce message. Cela permet d'assurer de l'intégrité des données.
  • Choisir un autre compte : ici, vous pouvez sélectionner quel compte vous voulez utiliser pour écrire des messages. Si vous avez activé la barre d'état (dans le menu Vue), le nom du compte courant est alors affiché en bas à droite de la fenêtre de Sylpheed (dans la barre d'état) et peut être changé en cliquant dessus (un menu contenant la liste de vos comptes apparaît).

6.7 Menu Aide

Ce menu vous permet d'accéder au manuel et à la FAQ de Sylpheed. Vous pouvez aussi voir des informations sur l'auteur et la version de Sylpheed dans le menu À propos.

6.8 Menu contextuel

La fenêtre des dossiers ainsi que celle du sommaire disposent chacune d'un menu contextuel que vous pouvez dérouler en cliquant sur le bouton 2 de la souris. (Utilisateurs de windows : le bouton 2 est le bouton droit de la souris.)

Dans la vue des dossiers, vous pouvez ajouter, renommer et enlever des dossiers par le menu contextuel. Vous pouvez aussi définir les propriétés des dossiers (par exemple définir le dossier sélectionné comme étant le dossier de Réception, d'envoi, poubelle, ...).

Dans le sommaire des messages, vous pouvez faire beaucoup de choses à partir du menu contextuel, comme voir les en-têtes du message, ouvrir un message dans une fenêtre séparée (c'est aussi possible en double cliquant sur un message), etc.
