3. Guide pour créer un nouveau compte pour groupes de discussion (news)

3.1 Obtenir les nouvelles dans l'urgence

Dans le menu de configuration, sélectionnez Création d'un nouveau compte....

Dans l'onglet de configuration Générale, entrez les données nécessaires : donnez un nom au compte, puis entrez votre nom et votre adresse email, etc. S'agissant d'un compte pour groupes de discussions, sélectionnez ensuite News (NNTP) dans le menu Protocole.

Puis, entrez le nom du serveur dans le champ Serveur de groupes de discussion.

Ignorez l'onglet Réception.

Dans l'onglet Envoi, vous pouvez configurer quelques informations supplémentaires.

Ensuite, dans l'onglet Composition, vous pouvez définir votre fichier de signature.

Dans l'onglet de configuration Avancée, vous pouvez éditer les numéros des ports des serveurs que vous allez utiliser. Mais vous n'avez normalement pas besoin d'y toucher.

Enfin, fermez les boîtes de dialogue, cliquez avec le bouton droit de la souris sur la nouvelle branche News dans l'arborescence des dossiers et sélectionnez Inscription aux groupes de discussion.... Si cela n'est pas déjà fait, une liste des groupes disponibles sera téléchargée du serveur. Arrivé à ce point, vous devriez vous débrouiller sans difficultés, bonne chance.
