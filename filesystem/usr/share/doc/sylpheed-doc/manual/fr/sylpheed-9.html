9. Composer des Messages

9.1 Composer un nouveau Message

Lorsque vous commencer à composer un nouveau message, une nouvelle fenêtre s'ouvre dans laquelle vous pouvez éditer le texte mais aussi spécifier les informations nécessaires à l'envoi. D'abord, la ligne De : vous présente une liste déroulante dans laquelle vous pouvez sélectionner le compte que vous voulez utiliser pour l'envoi.

Dans les parties À :,Cc : et Cci :, vous pouvez spécifier les adresses email respectives des destinataires principaux, de ceux qui recevront une copie et de ceux qui recevront une copie mais dont les adresses ne seront pas visibles pour les autres destinataires. Dans chaque champ, vous pouvez entrer une ou plusieurs adresses séparées par des virgules. Un outil important ici est le bouton Adresse qui permet d'accéder au carnet d'adresses. Dans la fenêtre de celui-ci, vous pouvez sélectionner une adresse et cliquer sur le bouton À :, Cc : ou Cci : pour la placer dans le champ destinataire correspondant. Le bouton Cci : peut être désactivé si ce champ n'apparaît pas dans la fenêtre de composition. Pour le rendre accessible, sélectionnez le menu Message -> Cci dans cette fenêtre.

Dans le champ du Sujet :, entrez un court texte descriptif sur le propos de votre message.

Vous pouvez vous déplacer parmi ces champs en pressant la touche de tabulation « TAB » (aussi désignée par « ->| ») pour aller directement à la ligne suivante ou « Shift-TAB » pour revenir en arrière. Enfin, pressez encore « TAB » dans le dernier champ et vous serez dans le corps du message prêt à l'éditer.

Si vous avez sélectionné un fichier de signature dans la configuration du compte utilisé, il sera déjà dans le corps du message. Vous pouvez alors écrire votre courrier. Lorsque c'est fait, vous pouvez soit cliquer sur le bouton Envoyer, soit sur le bouton Envoi différé, soit sur le bouton Brouillon dans la barre d'outils. Envoyer tente d'envoyer le message immédiatement. Envoi différé le placera dans le dossier File d'attente et enfin Brouillon le placera dans le dossier Brouillon.

9.2 Répondre à un message

Pour répondre à un message, utilisez la fonction Répondre. Dans Sylpheed, vous l'activez soit par le bouton Répondre soit en pressant « Alt-R ».

Lors d'une réponse, une nouvelle fenêtre de composition est créée avec les champs du destinataire et du sujet déjà remplis pour vous. La plupart des fonctionnalités décrites dans la section « Composer un nouveau message » sont aussi applicables à cette fenêtre.

9.3 Répondre en citant le message

Si vous voulez que Sylpheed cite le message auquel vous êtes en train de répondre, allez dans le menu de Configuration -> Configuration générale, puis dans l'onglet Composition validez l'option Citer le message en répondant. Vous pouvez aussi y définir le préfixe de la citation (la marque mise devant chaque ligne citée dans le courrier de réponse) ainsi que le texte introduisant la citation.

9.4 Transférer un message

Transférer un message (reçu) signifie envoyer à une tierce personne le message au complet y compris les en-têtes (l'adresse de l'expéditeur, la date d'envoi, etc.). Pour cela, pressez le bouton Transférer ou utilisez la combinaison des touches « Ctrl-F ». Une nouvelle fenêtre de message est créée contenant le texte du message sélectionné. Vous pouvez alors fournir le(s) nom(s) des destinataires du message, de la même manière que pour la composition d'un nouveau message. Vous pouvez même ajouter quelques lignes avant de l'envoyer.
