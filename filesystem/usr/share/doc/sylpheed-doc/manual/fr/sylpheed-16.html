16. Référence

Dans cette section, vous allez trouvez quelques informations sur raccourcis clavier utilisables dans Sylpheed et sur le format interne du fichier du carnet d'adresses.

16.1 Liste des raccourcis clavier

Raccourcis à une touche

  • a : crée un nouveau message ;
  • d : supprime le message sélectionné ;
  • f : fait suivre le message sélectionné ;
  • g : aller dans un dossier spécifique sélectionné par le biais d'une fenêtre dialogue ;
  • n : aller au message suivant / à la partie suivante d'un message contenant plusieurs parties ;
  • p : aller au message précédent / à la partie précédente d'un message contenant plusieurs parties ;
  • v : active / désactive la vue de message ;
  • w : crée un nouveau message (pareil que a) ;
  • y : sauvegarde le message dans un fichier.

Raccourcis du menu Fichier

  • Alt-F : ouvre le menu Fichier ;
  • Alt-P : imprimer ;
  • Alt-W : fermer ;
  • Alt-Q : quitter.

raccourcis du menu Éditer

  • Alt-E : ouvre le menu Éditer ;
  • Ctrl-C : copier ;
  • Ctrl-A : tout sélectionner ;
  • Ctrl-S : chercher.

Raccourcis du menu Vue

  • Alt-V : ouvre le menu Vue.

Raccourcis du menu Message

  • Alt-M : ouvre le menu Message ;
  • Alt-I : relever le courrier du compte actuellement utilisé ;
  • Shift-Alt-I : relever le courrier de tous les comptes ;
  • Alt-N : composer un nouveau message ;
  • Alt-R : répondre au message sélectionné ;
  • Ctrl-Alt-R : répondre à l'expéditeur (contrairement au Répondre [au message] décrit précédemment qui lui, peut répondre à une liste de diffusion, par exemple) ;
  • Shift-Alt-R : répondre à tous (à toutes les adresses situées dans les en-têtes « À : » et « Cc : ») ;
  • Ctrl-F : faire suivre le message sélectionné ;
  • Shift-Ctrl-F : faire suivre le message sélectionné comme pièce jointe ;
  • Alt-O : déplacer le message dans un autre dossier ;
  • Alt-D : supprimer le message ;
  • Shift-Ctrl-N : ouvrir le message dans une nouvelle fenêtre ;
  • Ctrl-U : voir le source du message ;
  • Ctrl-H : montrer tous les en-têtes du message.

Raccourcis du menu Sommaire

  • Alt-S : ouvre le menu Sommaire ;
  • Alt-X : exécute les commandes en attente, uniquement utile lorsque les commandes « supprimer », « déplacer » et « copier » ne sont pas exécutées immédiatement ;
  • Alt-U : met à jour des vues du sommaire et du message ;
  • Alt-G : va dans un autre dossier ;
  • Ctrl-T : active la vue par fils de conversation ;
  • Shift-Ctrl-T : désactive la vue par fils de conversation.

Raccourcis du menu Outils

  • Alt-T : ouvre le menu Outils ;
  • Alt-A : ouvre le carnet d'adresses ;
  • Alt-L : ouvre le journal de connexion (log).

16.2 Format de données du carnet d'adresses

Le carnet d'adresses est situé dans le répertoire ~/.sylpheed. Il est nommé addressbook.xml. Le format général du carnet d'adresses est le suivant : 

< addressbook ><p>
 < common_address ><p>
    < group name="Common addresses" ><p>
    < item ><p>
     < name > Paul < /name ><p>
     < address > paul@nlpagan.net < /address ><p>
     < remarks > Écrit le manuel de Sylpheed < /remarks ><p>
        < /item ><p>
    < /group ><p>
 < /common_address ><p>
< /addressbook >
